Projeto Bingo Ágil (Atividade 4)

Este projeto é uma implementação de um jogo de Bingo em Java, focado na aplicação de conceitos de Design Ágil e Clean Code.

Sobre o Projeto
Disciplina: Processos de Design Ágil de Software  
Aluna: Melissa Brito
Objetivo: Desenvolver um MVP (Produto Mínimo Viável) de um jogo de bingo com cartela de 24 números.

Roadmap do Produto

O desenvolvimento deste produto segue o seguinte planejamento evolutivo:

Versão 1.0 (Atual - MVP):
    [x] Sorteio aleatório de números.
    [x] Cartela com 24 números (Regra de negócio).
    [x] Interface via Console.
    [x] Validação automática de vitória.

Versão 2.0 (Próxima Sprint):
    [ ] Regras oficiais de colunas (B-I-N-G-O).
    [ ] Persistência de dados (Salvar histórico).
    
Versão 3.0 (Futuro):
    [ ] Interface Gráfica (GUI).
    [ ] Modo Multiplayer Online.

Aplicação do Lean

Utilizou-se o princípio Lean de "Eliminar Desperdícios".
Optou-se por desenvolver um MVP em modo texto para validar a regra de negócio (sorteio e cartela) rapidamente, evitando o desperdício de recursos na criação prematura de interfaces gráficas complexas antes da validação da lógica central.

Como Rodar
1. Clone este repositório.
2. Abra no VS Code.
3. Execute o arquivo `src/bingo/BingoGame.java`.
